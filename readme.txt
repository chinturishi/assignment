->To run this cucumber framework below are prerequisites

1. Java 1.8 or above.
2. Eclipse IDE 2019-06.
3. Browser Google Chrome should present in the PC.
4. Download the Maven and configure in the PC.

How to run

1. Clone the repo by using the below link https://github.com/chinturishi/assignment.git
2. Copy the projcet to your work space.
3. Open the project in the eclipse.
4. Right click on the project and go to Run As Maven Clean, after that do Maven test.
5. After the test finished you can see the report in the below path "Cucumber_framework\target\Extent_Reports"
